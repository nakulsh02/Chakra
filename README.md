# Chakra

Chakra is an interactive web project showcasing **Chakra-inspired visuals** and animations.  
The project focuses on a **modern, clean design**, smooth transitions, and responsive layout.

## Features

- Interactive Chakra visuals
- Smooth animations and effects
- Modern, clean user interface
- Fully responsive on mobile, tablet, and desktop
- Runs directly in the browser
